Terms

Data types

typeof to find out what data type

  • Numeric - number
  • String - text
  • Boolean - true or false
  • Null - nothing, nonexistent or invalid
  • Object - Everything
  • Undefined
  • Symbol
  • Array - a bag that you can throw things into []

Comparison Operators

  • = set variable
  • == equal to
  • === strict equal to
  • != not equal to
  • !== strict not equal to
  • > greater than
  • >= greater than or equal to
  • < less then
  • <= less than or equal to
  • && logical and (has to be both)
  • || logical or (has to be one)
  • ! logical not (inverts value)
    !(2 > 1) returns true

Selectors

  • getElementById('star') - only ID
  • getElementByClassName('star') - only class
  • getElementByTagName('P') - only tags
  • querySelector('#star') - only the first one of id/class/tag
  • querySelectorAll('.star') - all with of id/class/tag, nodeList instead of HTMLcollection

Functions

  • (function(){})(); Iffe - immediately invoked
  • function declaration () {}
  • var expression = function () {}
